📞 SUPPORT:

  → Install: choco install tesseract (Windows)
  → Install: sudo apt-get install tesseract-ocr (Linux)
  → Install: brew install tesseract (Mac)

  → Change port in backend/app.py: app.run(port=5001)

Problem: Module import error
  → Run: pip install -r requirements.txt

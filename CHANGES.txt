🎯 HOW TO RUN:

LOCAL DEVELOPMENT:
  1. Windows: Double-click start.bat (automatic setup + launch)
  2. Linux/Mac: 
     python3 -m venv venv
     source venv/bin/activate
     pip install -r requirements.txt
     python -m textblob.download_corpora
     python backend/app.py
  
  3. Open browser:
     http://localhost:5000 (for testing API)
     frontend/index.html (for UI - auto-connects to localhost:5000)

DEPLOY TO VERCEL:
  1. git add . && git commit -m "Deploy" && git push
  2. Go to vercel.com/new and import GitHub repo
  3. Click Deploy
  4. Access at https://your-project.vercel.app

DEPLOY TO RENDER (Docker):
  1. Connect GitHub repo to Render
  2. Select "Web Service"
  3. Deploy from render.yaml

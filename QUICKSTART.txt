🔧 TROUBLESHOOTING:

Problem: "Port 5000 already in use"
  → Change port in backend/app.py: app.run(port=5001, ...)

Problem: "Tesseract not found"  
  → Windows: Install Tesseract: choco install tesseract
  → Linux:   sudo apt-get install tesseract-ocr
  → Mac:     brew install tesseract

Problem: "Module not found"
  → Run: pip install -r requirements.txt

Problem: "EasyOCR slow first time"
  → Normal! Models (~350 MB) download automatically
  → Next requests will be much faster

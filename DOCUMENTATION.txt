QUICK REFERENCE:

LOCAL DEVELOPMENT:
  • Windows: Double-click start.bat
  • Linux/Mac: python backend/app.py
  • Frontend: Open frontend/index.html
  • Backend: http://localhost:5000

DEPLOYMENT:
  • Vercel: Go to vercel.com/new → Select GitHub repo → Deploy
  • Render: Connect GitHub → Deploy from render.yaml
  • Docker: docker build -t ocr . && docker run -p 5000:5000 ocr

TESTING:
  • python test_ocr.py        (test OCR endpoint)
  • python verify_deps.py     (check dependencies)
  • python check_system.py    (full verification)

API QUICK REFERENCE:

GET  /api/health              Get server status
POST /api/ocr                 Process image
GET  /api/history             Get scan history
GET  /api/history/search      Search scans
DELETE /api/history/{id}      Delete scan
POST /api/tts                 Text-to-speech
POST /api/export              Export results

See README.md for full API documentation with examples.
